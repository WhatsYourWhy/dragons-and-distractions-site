## Feedback welcome

If you spot a broken link, a monster that sends you to the wrong next step, a printable that does not print cleanly, or a page that feels confusing on a low-energy day, that feedback is useful.

For feedback, partnership questions, or licensing inquiries:

`a689569 [at] gmail [dot] com`

If you write in, the most helpful note is:

- the page URL
- what you expected
- what happened instead
- device or browser if it looks like a display issue

## Privacy notes

This is a static site hosted on GitHub Pages.

- There are no user accounts.
- There are no on-site comments or forms.
- Two browser-side preferences are stored in `localStorage`:
  - `dd-theme` — your low-stim theme choice.
  - `dd-consent` — whether you have accepted or declined analytics via the consent banner.
- Analytics: we use Google Analytics 4 (measurement ID `G-4BBWE83NT3`) with IP anonymisation. The analytics script only loads after you click **Accept** on the consent banner; declining keeps it off. You can reset your choice any time by clearing `dd-consent` from your browser's site storage.
- We do not load Google AdSense or any other ad network on the site.
- If additional tracking is added later, this page will be updated before or at the same time.

## Support boundary

This project is a practical companion, not therapy, crisis support, diagnosis, or medical care.

- Use the toolkit for recognition, gentler self-understanding, and one workable next step.
- If you are in crisis or need urgent mental health support, use local emergency or crisis resources and qualified care.
- If a page stops being helpful, you do not need to force the metaphor to fit.
