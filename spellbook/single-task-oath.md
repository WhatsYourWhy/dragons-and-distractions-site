  <p class="field-guide__lead">Use this ritual when Task Hydra has turned one job into a field of active heads. The point is not to become perfectly focused. The point is to create one honest lane and stay in it.</p>

  <div class="field-guide__grid">
    <section class="field-guide__card field-guide__card--quote">
      <h2>The oath</h2>
      <div class="field-guide__quote">
        <p>I choose one head. I do not grow new ones.</p>
        <p>Today's three quests are visible. The rest can wait.</p>
      </div>
    </section>

    <section class="field-guide__card">
      <h2>How to use it</h2>
      <ol>
        <li>Name one task, not an entire project category.</li>
        <li>Gather only the tools that belong to that task.</li>
        <li>Put every new idea in a parking lot instead of promoting it into the current battle.</li>
        <li>Finish, stop, and choose again.</li>
      </ol>
    </section>

    <section class="field-guide__card">
      <h2>Why it helps</h2>
      <ul>
        <li>It reduces switching costs.</li>
        <li>It creates an external rule before overwhelm starts negotiating.</li>
        <li>It keeps new ideas from hijacking the current lane.</li>
      </ul>
    </section>
