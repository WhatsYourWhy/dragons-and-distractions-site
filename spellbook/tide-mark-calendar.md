    <section class="field-guide__card">
      <h2>Setup steps</h2>
      <ol>
        <li>Pick a real start time, not just a due date.</li>
        <li>Add a T-minus 60 alarm for setup and a T-minus 15 alarm for opening the first bite.</li>
        <li>Stage the first bite before you walk away.</li>
        <li>Put the start time somewhere visible.</li>
        <li>If needed, tell a real person when the start happens.</li>
      </ol>
    </section>

    <section class="field-guide__card">
      <h2>Why it helps</h2>
      <ul>
        <li>It turns intention into a visible commitment.</li>
        <li>It reduces the gap between "I should" and "I am starting now."</li>
        <li>It gives the brain something concrete before panic takes over.</li>
      </ul>
    </section>

    <section class="field-guide__card field-guide__card--quote">
      <h2>Use it when</h2>
      <div class="field-guide__quote">
        <p>The deadline feels real only once it is already urgent.</p>
        <p>You need the start line to become visible before panic tries to do the job for you.</p>
      </div>
    </section>
