    <section class="field-guide__card field-guide__card--quote">
      <h2>The oath</h2>
      <div class="field-guide__quote">
        <p>I will name what must happen. Everything else is optional until proven otherwise.</p>
        <p>One recovery action goes on the list before a single new task does.</p>
      </div>
    </section>

    <section class="field-guide__card">
      <h2>How to use it</h2>
      <ol>
        <li>List only what truly must happen today. If it can wait until tomorrow without real consequences, it waits.</li>
        <li>Add one recovery action to the list: a nap, a walk, a meal, silence. It is not optional.</li>
        <li>Cross off anything you added from habit, guilt, or fantasy fuel.</li>
        <li>Do the surviving list in any order. Stop when it is done.</li>
      </ol>
    </section>

    <section class="field-guide__card">
      <h2>Why it helps</h2>
      <ul>
        <li>It matches the plan to the actual fuel supply instead of yesterday's capacity.</li>
        <li>It protects recovery from being the first thing cut.</li>
        <li>It lets you finish a day instead of failing a bigger one.</li>
      </ul>
    </section>

    <section class="field-guide__card field-guide__card--quote">
      <h2>Use it when</h2>
      <div class="field-guide__quote">
        <p>Simple things now feel steep, and rest keeps getting postponed because there is always more to carry.</p>
        <p>You need a smaller day, not a more inspiring plan.</p>
      </div>
    </section>
